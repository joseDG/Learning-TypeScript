### Curso de TypeScript

#### Instalacion typescript forma global
```
npm install -g typescript
```
#### Revisar la version de typeScript
```
tsc --version
```
#### Para recorrer un programa se usa.
```
tsc app.ts
```
#### Para poner en modo watch.
```
tsc app.ts -w

tsc *.ts -w
 ```
 #### Para iniciar un proyecto se usa.
```
tsc -init
 ```
